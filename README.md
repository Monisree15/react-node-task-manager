# Full Stack Task Manager (React + Node.js + MySQL)

A simple full-stack task management application built using React, Node.js, Express, and MySQL. 
This project demonstrates REST API development, frontend–backend integration, and relational database usage.

## Features

- Add new tasks  
- View all tasks  
- Delete tasks  
- RESTful APIs  
- MySQL database integration  

## Tech Stack

Frontend: React, JavaScript, HTML, CSS  
Backend: Node.js, Express  
Database: MySQL
